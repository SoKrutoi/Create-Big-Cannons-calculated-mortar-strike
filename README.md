# CBC — Ballistic Computer

Программа для расчёта направления и параметров наведения пушки или миномёта.

<sup>За основу взят https://github.com/Malex21/CreateBigCannons-BallisticCalculator/</sup>

## Есть html веб версия она даже удобнее, так что лучше использовать её

## Установка

1. Скачайте или клонируйте репозиторий.
2. Установите зависимость:

```bash
pip install customtkinter
```

## Запуск

```bash
python gui.py
```

## Использование

### 1. Выберите тип орудия

В верхней части окна переключите **Пушка** или **Миномёт**.

Для **пушки** укажите:
- тип снаряда (в текущей версии выбор снаряда не влияет на расчёт);
- количество пороха (шт);
- длину ствола (блоки) — расстояние от места крепления пушки до кончика ствола.

Для **миномёта** дополнительные параметры не требуются.

### 2. Введите координаты

- **Позиция Орудия (X;Y;Z)** — координаты крепления пушки или основания миномёта.
- **Позиция Цели (X;Y;Z)** — координаты точки попадания.

Программа автоматически пересчитывает траекторию при изменении любого поля, если все поля заполнены.

### 3. Получите решение

После нажатия **«Рассчитать Траекторию»** отобразятся:

- **YAW** — угол поворота орудия (горизонталь);
- **V0** — начальная скорость снаряда;
- **Pitch (Настильная)** — угол для прямой траектории;
- **Pitch (Навесная)** — угол для навесной траектории;
- **Время полёта** для каждой траектории.

### 4. Настройте орудие

Используя вращение и передатчик информации, установите рассчитанные Pitch и Yaw. Затем поместите снаряд в ствол и произведите выстрел.

## Графики

- **Боковой вид (X-Y)** — траектория полёта снаряда. Зелёная линия — настильная, оранжевая — навесная.
- **Вид сверху (X-Z)** — положение орудия и цели в абсолютных мировых координатах, линия наведения и зона разброса. Кнопка **«Зона»** в шапке графика включает оверлей досягаемости.

На обоих графиках работает перетаскивание мышью и масштабирование колёсиком.

## Калибровка

Если расчёты не совпадают с реальными попаданиями в вашем модпаке, перейдите на вкладку **«Калибровка»**:

1. Выберите тип орудия.
2. Введите координаты орудия, координаты фактического попадания, углы выстрела (Pitch, Yaw). Для пушки также укажите порох и длину ствола.
3. Опционально укажите время полёта — это повысит точность.
4. Нажмите **«Добавить точку»**. Повторите для нескольких выстрелов.
5. Нажмите **«Запустить калибровку»**.

Программа подберёт физические константы и сохранит их в `calibration_config.json`.
