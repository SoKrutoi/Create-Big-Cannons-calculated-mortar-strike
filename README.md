# Миномётный калькулятор

<img width="480" height="252" alt="Результат" src="https://github.com/user-attachments/assets/2dccc225-8fc3-4981-aeb8-ea11915a70f2" />

Программа для расчёта направления и параметров наведения миномётной пушки.
- (Максимальный радиус миномётной пушки ~1500 блоков)
## Установка

1. Скачайте или клонируйте репозиторий.
2. Распакуйте проект в любую папку (если скачивали архив).

## Запуск

Требуется **Python 3.10+**.

```bash
python gui.py
```

## Использование

### 1. Постройте миномётную пушку

<img width="485" height="345" alt="Миномётная пушка" src="https://github.com/user-attachments/assets/25b50b60-967d-4441-a45d-8810f06c9476" />

### 2. Введите координаты

Введите:
- координаты крепления пушки;
- координаты цели.

Нажмите **«Рассчитать»**.

### 3. Настройте пушку

Используя **вращение** и **передатчик информации**, установите рассчитанные значения:
- Pitch (Наклон вверх);
- Yaw (Поворот влево вправо).

После этого поместите в ствол **миномётный снаряд со взрывателем** и произведите выстрел.
